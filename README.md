# Universal Helper Platform v1.0

Универсальный игровой помощник для **GTA San Andreas — SA-MP** (сервера Advance RP).  
Построен на **MoonLoader** (Lua) с GUI **mimgui**, событиями **SAMP.Lua** и **SAMPFUNCS**.

[English version](README_EN.md) | [Инструкция по эксплуатации](INSTRUCTION_RU.md)

---

## Возможности

### AutoEdit — Автоматическое форматирование объявлений (NEW v1.0)
- Перехват диалога `/edit` через `sampev.onShowDialog`
- Вместо стандартного диалога открывается **ImGui окно** с:
  - Оригинальным текстом объявления
  - Отформатированным результатом (зелёным)
  - Полем редактирования — можно подправить текст
- **800+ правил автозамены** сленга на правильные названия:
  - Машины: булка→а/м марки "Bullet", инф→"Infernus", туризмо→"Turismo", султан→"Sultan", хотринг→"Hotring" и др. (все модели GTA SA + реальные бренды)
  - Мото: нрг→мото марки "NRG-500", санч→"Sanchez", пжж→"PCJ-600"
  - Города: лс→Los Santos, сф→San Fierro, лв→Las Venturas
  - Районы: гетто/ждлс→East Los Santos, гантон, идл, верон, элкорона, флинт кантри, паломино крик, форт карсон, тиера робада, бэйсайд и др.
  - Недвижимость: дом→жилье (дом, квартиру, особняк, времянку), биз→бизнес, качалка→тренажёрный зал, АЗС, казино, отель, магазин и др.
  - Предметы: сим→SIM-card, меч→аксессуар "Меч", оружие, одежда, рюкзак, часы и др.
  - Оружие: дигл→"Desert Eagle", м4→"M4", ак→"AK-47" и др.
  - Работа: сми→СМИ, мэрия→мэрию, собеседование, вакансия и др.
- **Тег сервера** — настраиваемый префикс (`LV | `, `LS | `, `SF | `, `TV | `)
- **Авто-цена** — если нет цены, добавляет "Цена: договорная"
- **Заглавная буква** — корректная капитализация кириллицы CP1251 после тега
- **Отправка по Enter** — не нужно нажимать кнопку мышью
- Очистка SAMP color codes (`{FFCC15}`, `{FFFFFF}`)

### Круиз-контроль
- **Обычный режим** — симулирует педаль газа с гистерезисом (плавно, надёжно)
- **Турбо-режим** — спидхак через `setCarForwardSpeed` с защитой:
  - Детекция удара (падение скорости > 40% → пауза 3 сек)
  - Плавный разгон (+3 → +20 за кадр)
  - Проверка двигателя (не разгоняет при заглушенном двигателе)
  - Проверка в воздухе (не применяет в полёте)
- **Управление:** `C` = вкл/выкл, `W` = +5, `S` = -5

### Стробоскопы (19 режимов)
- Управление фарами через память (`CDamageManager`)
- Режимы: мигающие, попеременные, бегущие, сирены, SOS, гирлянда, маяк, патруль и др.
- **Управление:** `J` = вкл/выкл, `N` = следующий режим

### Система биндов клавиш
- Привязка любой клавиши к любой команде чата
- По умолчанию: `L` = `/lock`, `K` = `/e`
- Добавление/редактирование/удаление через интерфейс
- Сохранение в `helper_settings.json`
- Не срабатывает при открытом чате или диалоге

### Авто-звонки и база данных
- Отслеживает игроков из чата, сохраняет ник + телефон
- Авто-звонок онлайн-игрокам из БД
- Настраиваемая задержка, максимум звонков и кулдаун (не звонить человеку N часов)
- История звонков сохраняется между сессиями
- Прогресс: сколько позвонили / осталось
- Сброс истории и очистка БД из интерфейса

### Авто-реклама
- Автоматическая публикация рекламы с настраиваемой задержкой
- Распознавание формата Advance RP: `Отправил Nick[32] (тел. 123)`
- Шаблоны объявлений

### MM Editor (СМИ Редактор)
- Смена скина игрока по ID модели (0–311)
- **AutoEdit** — автоматическое форматирование объявлений (см. выше)
- Настраиваемый тег сервера (LV, LS, SF, TV)
- Авто-форматирование при открытии редактора `/edit`
- Авто-отправка с настраиваемой задержкой
- Тест автозамены прямо в интерфейсе

### Авто-РП
- Автоматизированные RP-отыгровки через `/me` перед серверными командами
- Команды: `/call`, `/h`, `/mask`, `/healme`, `/drugs`, `/e`
- Работает через перехват `sampev.onSendChat` (не блокирует серверные команды)

### Сканер фракций
- Определяет фракцию из диалога, автоматически ставит скин

### Фиксация погоды и времени
- Блокировка серверных пакетов (`onSetPlayerTime`, `onSetWeather`)
- Заморозка внутреннего таймера GTA SA
- Управление: ползунки ID погоды (0-45) и часа (0-23) в интерфейсе

### Сохранение настроек
- Состояния модулей сохраняются между перезагрузками (исправлено в v1.0)
- Тег сервера, бинды, история звонков, шаблоны рекламы — всё в `helper_settings.json`

### Справочник команд
- Встроенный справочник по командам Advance RP по категориям

## Установка

Подробная инструкция: [INSTRUCTION_RU.md](INSTRUCTION_RU.md)

Кратко:
1. Установите **CLEO 4** (обязательно для SAMPFUNCS)
2. Установите **MoonLoader v0.26+** — качайте **полную сборку** (со всеми библиотеками)
   (полная сборка уже содержит mimgui, SAMP.Lua, memory, bit, encoding, json, ffi)
3. Установите **SAMPFUNCS** (в папку с игрой)
4. Скачайте **`helper_core.lua`**: https://github.com/estatyq/advancelua/raw/main/helper_core.lua
5. Скопируйте в `GTA San Andreas/moonloader/`
6. Запустите игру, нажмите **F11** или введите `/helper`

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `F11` | Открыть/закрыть главное меню |
| `C` | Вкл/выкл круиз-контроль (в транспорте) |
| `W` / `S` | Увеличить / уменьшить целевую скорость круиза |
| `J` | Вкл/выкл стробоскопы (в транспорте) |
| `N` | Следующий режим стробоскопов (в транспорте) |
| `L` | `/lock` (закрыть/открыть машину) |
| `K` | `/e` (завести/заглушить двигатель) |

Все горячие клавиши отключены при открытом чате или диалоге.

## Использование AutoEdit (объявления)

1. Откройте меню **F11** → модуль **MM Editor (СМИ)** → активируйте
2. В поле **"Тег объявления"** введите ваш тег (например `LV`, `LS`, `SF`, `TV`)
3. Включите **"Авто-форматирование при открытии редактора"**
4. На сервере введите `/edit` — откроется окно AutoEdit вместо стандартного диалога
5. В окне вы увидите:
   - **Оригинальный текст** (серым)
   - **Отформатированный результат** (зелёным) — с заменами и тегом
   - **Поле редактирования** — предзаполнено результатом, можно править
6. Нажмите **Enter** или кнопку **"Отправить"** — объявление уходит на сервер
7. Нажмите **"Отмена"** — закрыть без отправки

**Примеры:**
| Ввод | Результат |
|------|-----------|
| `продам булку дог` | `LV \| Продам а/м марки "Bullet". Цена: договорная` |
| `куплю дом в лс` | `LV \| Куплю жилье (дом, квартиру, особняк, времянку) в Los Santos. Цена: договорная` |
| `продам хотринг 5кк` | `LV \| Продам а/м марки "Hotring" 5.000.000$` |
| `семья bob ищет родственников` | `LV \| Семья bob ищет родственников` |

**Авто-отправка:** если включена — скрипт сам отправит объявление через задержку. Если выключена — просто заполнит поле, вы жмёте Enter.

## Настройки

Настройки сохраняются в `moonloader/config/helper_settings.json`:
- Включение/выключение модулей (сохраняется между перезагрузками)
- Тег сервера для объявлений
- Авто-форматирование и авто-отправка
- Режим и скорость стробоскопов
- Турбо-режим круиз-контроля
- Пользовательские бинды клавиш
